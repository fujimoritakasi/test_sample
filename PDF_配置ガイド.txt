# PDFファイル配置ガイド

このフォルダに以下の名前でPDFファイルを配置してください：

## 📄 必要なPDFファイル

1. **document.pdf** - 表示したいPDFドキュメント
   - 任意の内容のPDFファイルを配置してください

## 📂 配置場所

PDFファイルは以下のパスに配置してください：
```
c:\OAU\test_sample\document.pdf
```

## 🔄 ファイル名を変更する場合

別の名前のPDFファイルを使用する場合は、`index.html` 内の以下の部分を編集してください：

```html
<a href="document.pdf" target="_blank" class="pdf-link">
```

この `document.pdf` の部分を実際のファイル名に変更してください。

## ✅ 動作確認

PDFファイルを配置した後、`index.html` をブラウザで開いて動作を確認してください：

1. PDFリンクをクリックして新しいタブでPDFが開くか
2. インライン表示でPDFが表示されるか
3. モバイルデバイスでも正常に動作するか

## 📝 注意事項

- PDFファイルのサイズが大きすぎると読み込みに時間がかかります
- ファイル名に日本語や特殊文字は使用しないことを推奨します
- PDFファイルが見つからない場合、404エラーが表示されます
